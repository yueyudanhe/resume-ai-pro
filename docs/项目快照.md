# ResumeAI Pro 项目快照

**更新时间**: 2026-03-15
**版本**: MVP v0.1.0

---

## 项目概述

AI简历优化工具，面向全球市场，支持15种语言，提供简历分析、评分和优化建议服务。

---

## 核心功能

### MVP (当前阶段)
- [x] 项目初始化
- [x] 文件上传组件（PDF/Word）
- [x] AI简历分析（Claude API）
- [x] 评分系统（免费显示总分+3条问题）
- [x] 完整报告（付费解锁）
- [x] Stripe支付集成
- [x] 多语言支持（15种语言自动检测）

### 后续版本
- [ ] 简历模板生成
- [ ] 历史记录
- [ ] 用户账户系统

---

## 技术栈

| 层级 | 技术 |
|-----|------|
| 框架 | Next.js 16 + React 19 |
| 语言 | TypeScript (strict: true) |
| 样式 | Tailwind CSS 4 |
| AI | Claude API (@anthropic-ai/sdk) |
| 支付 | Stripe |
| 文件解析 | pdf-parse, mammoth |
